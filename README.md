## Aluba
A high-converting restaurant website for Àbulà Central, Ikoyi, Lagos. Built with HTML, CSS, and JavaScript.

## Features
- SVG hero wordmark replicates the hand-lettered signage font from the physical restaurant
- Full-bleed food photography real restaurant images, not stock photos
- Scroll animations elements reveal on scroll using IntersectionObserver
- Google Maps embed real address pinned at 19 Alfred Rewane Rd, Ikoyi
- SVG icons only no icon fonts, no external icon libraries
- Zero dependencies no npm, no build step, no frameworks
- Fully responsive works on mobile, tablet, and desktop
